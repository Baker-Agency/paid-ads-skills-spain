# Unit Economics — X Ads Espana

## Framework CAC para X Ads

### Formula base

```
CAC = (Gasto publicitario + Fee gestion) / Clientes conseguidos
```

### Calcular con CPC

1. CPC medio × 100 = gasto por 100 clics
2. CVR landing × 100 = leads
3. Tasa cierre × leads = clientes
4. Gasto total / clientes = CAC

### Ejemplo: SaaS B2B Espana

| Paso | Metrica | Valor |
|---|---|---|
| CPC medio | | €1,50 |
| Gasto 100 clics | €1,50 × 100 | €150 |
| CVR landing 2% | 100 × 0,02 | 2 leads |
| Tasa cierre 25% | 2 × 0,25 | 0,5 clientes |
| CAC | €150 / 0,5 | **€300** |

Con fee gestion €300/mes: (€150 + €300) / 0,5 = **€900 CAC real** (para 100 clics)

## IVA 21% — incluir siempre

- Revenue reportado por cliente suele ser **IVA incluido**
- Al calcular margen real: Revenue / 1,21 = Revenue neto
- Ejemplo: cliente paga €121 → revenue real = €100
- **Todo calculo de ROI debe usar revenue neto (sin IVA)**

## Cuando X Ads NO tiene sentido

- LTV cliente < €200 y CVR < 2% → CAC demasiado alto
- Audiencia objetivo +55 anos → baja presencia en X
- Producto visual que necesita Discovery → Meta es mejor
- Lead gen local puro → Google Ads es mejor
