# Pujas y Presupuesto — X Ads Espana

## Estrategias de puja

### Automatica (recomendada al inicio)

- X optimiza pujas para maximizar acciones dentro del presupuesto
- **Mejor para:** campanas nuevas, fase de aprendizaje, sin datos historicos
- **Ventaja:** no necesitas saber CPC/CPA optimo
- **Riesgo:** puede gastar rapido si no hay cap de presupuesto diario

### Puja maxima (Manual)

- Tu defines el maximo que pagas por accion
- **Mejor para:** campanas maduras, control de costes, optimizacion fina
- **Ventaja:** control total de CPC/CPA
- **Riesgo:** si fijas demasiado bajo → poca entrega

### Puja objetivo (Target)

- Defines el coste medio por accion deseado
- X ajusta pujas en tiempo real para alcanzar ese promedio
- **Mejor para:** cuando tienes benchmark claro de CPA/CPC objetivo
- **Ventaja:** equilibrio entre control y volumen
- **Riesgo:** puede fluctuar dia a dia

## Progresion de pujas recomendada

```
Automatica (0-30 dias) → Target (30-60 dias con datos) → Manual (60+ dias, optimizacion fina)
```

### Reglas de transicion

| De | A | Cuando |
|---|---|---|
| Automatica | Target | 30+ conversiones y CPA estable |
| Target | Manual | CPA consistente 2+ semanas, quieres bajar coste |
| Cualquiera | Automatica | Cambio grande de audiencia o creativo (resetear datos) |

**Regla de oro:** esperar 3-7 dias despues de cualquier cambio de puja antes de evaluar.

| Modelo | Paga por | Mejor para | CPC/CPX tipico |
|---|---|---|---|
| **CPC** | Clic en enlace | Lead gen, trafico web | €0,45-1,80 |

## Presupuesto: tipos

### Presupuesto diario

- Limita gasto cada 24 horas
- **Delivery Standard:** distribuye gasto uniformemente durante el dia
- **Delivery Accelerated:** gasta lo mas rapido posible (usar solo para eventos/urgencias)
- **Recomendado para:** mayoria de campanas

### Presupuesto total (lifetime)

- Maximo acumulado para toda la duracion de campana
- X distribuye automaticamente entre los dias
- **Recomendado para:** promociones con fecha limite, eventos

## Presupuesto recomendado para Espana

### Por nivel de inversion

| Nivel | Mensual | Diario | Que puedes hacer |
|---|---|---|---|
| **Micro-test** | €300-500 | €10-17 | Validar 1 audiencia, 1 formato. Datos limitados |
| **Testing serio** | €500-1.000 | €17-33 | 2-3 audiencias, A/B testing basico |
| **Escala inicial** | €1.000-3.000 | €33-100 | Lead gen activo, retargeting, 2-3 campanas |
| **Escala media** | €3.000-10.000 | €100-333 | Multi-campana, video, full funnel |
| **Enterprise** | €10.000+ | €333+ | Takeovers, Amplify, brand + performance |

### Presupuesto minimo por objetivo

| Objetivo | Minimo diario recomendado |
|---|---|
| Awareness | €10-20 |
| Trafico web | €20-30 |
| Video views | €15-25 |
| Lead gen (CPA) | €30-50 |
| App installs | €30-50 |

**Regla:** si no alcanzas 10-15 acciones/dia con tu presupuesto, la campana no tiene datos suficientes para optimizar.

## Distribucion de presupuesto

### Formula 70/20/10

| Porcentaje | Destino |
|---|---|
| **70%** | Campanas probadas que funcionan (core) |
| **20%** | Escalado de campanas prometedoras |
| **10%** | Testing de nuevos formatos/audiencias |

### Entre campanas

- **Prospecting:** 60-70% del presupuesto
- **Retargeting:** 20-30% del presupuesto
- **Brand/Awareness:** 10% (si aplica)

## Factores que afectan al coste

| Factor | Efecto en coste |
|---|---|
| Competencia (mas anunciantes = mas caro) | ↑ CPC/CPM |
| Calidad creativa (engagement predicho alto) | ↓ CPC/CPM |
| Historial de cuenta (buen rendimiento previo) | ↓ CPC/CPM |
| Estacionalidad (Black Friday, Navidad) | ↑ CPC/CPM 20-50% |
| Audiencia muy estrecha | ↑ CPM |
| Audiencia muy amplia sin exclusiones | ↑ CPC (clics irrelevantes) |

## Pacing — controlar el ritmo de gasto

- **Revisar diariamente** los primeros 7 dias
- Si gasta demasiado rapido (>120% del diario): reducir presupuesto o estrechar audiencia
- Si gasta poco (<50% del diario): ampliar audiencia, subir puja, o mejorar creativo
- **Delivery Standard** = mas consistente. Usar siempre excepto urgencias
