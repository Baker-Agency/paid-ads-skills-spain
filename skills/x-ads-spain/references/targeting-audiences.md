# Targeting y Audiencias — X Ads Espana

## Tipos de segmentacion

### 1. Demografica

| Parametro | Opciones |
|---|---|
| Ubicacion | Pais → Region → Ciudad → Metro |
| Edad | Rangos (18-24, 25-34, 35-49, 50-64, 65+) |
| Genero | Hombre, Mujer, Cualquiera |
| Idioma | Espanol, Catalan, Ingles, etc. |
| Dispositivo | iOS, Android, Desktop |
| SO | Version especifica si necesario |

**Para Espana:**
- Ubicacion base: Espana
- Si presupuesto permite: segmentar por CC.AA. (Madrid, Barcelona, Valencia = 50%+ audiencia)
- Edad recomendada: 25-54 (nucleo activo en X)
- Idioma: Espanol

### 2. Keyword targeting (diferenciador clave)

**Que es:** segmentar por palabras que usuarios han buscado, tuiteado o con las que han interactuado en los ultimos 7 dias.

**Por que es unico:** ninguna otra plataforma permite segmentar por conversacion activa del usuario.

**Como usar:**

1. **Identificar keywords de intencion:**
   - "necesito CRM" → intencion de compra
   - "mejor software de..." → investigacion activa
   - "problemas con [competidor]" → oportunidad de captura

2. **Keywords de conversacion:**
   - "marketing digital" → interes general
   - "automatizar procesos" → pain point
   - "escalar negocio" → ambicion/crecimiento

3. **Negative keywords (obligatorias):**
   - "gratis", "free", "crack" → eliminar buscadores de gratuito
   - "empleo", "trabajo", "sueldo" → eliminar buscadores de empleo
   - "tutorial", "curso" (si no vendes formacion) → eliminar estudiantes

**Ejemplo practico — SaaS CRM Espana:**

| Keywords positivas | Keywords negativas |
|---|---|
| crm espanol | gratis |
| gestion clientes | tutorial |
| automatizar ventas | empleo |
| pipeline ventas | excel plantilla |
| software comercial | open source |
| herramienta seguimiento leads | comparativa (si no quieres top-funnel) |

**Volumen recomendado:** 25-50 keywords por grupo de anuncios. Ni demasiado pocas (poco volumen) ni demasiadas (pierde precision).

### 3. Intereses

350+ categorias en 25 temas principales. Los mas relevantes para lead gen Espana:

| Categoria | Subcategorias utiles |
|---|---|
| Technology | Software, SaaS, Startups, AI |
| Business | Entrepreneurship, Small business, Marketing |
| Finance | Investing, Personal finance, Fintech |
| Education | Online learning, Professional development |
| Health | Clinics, Wellness, Fitness |
| Careers | Job search, Professional growth |

**Combinar intereses + keywords** = audiencia mas precisa.

### 4. Conversation topics

Temas activos de conversacion en tiempo real. Util para:
- Eventos del sector (MWC Barcelona, South Summit)
- Temas trending relevantes
- Temporadas especificas (Black Friday, vuelta al cole)

### 5. Follower look-alikes

Targetear usuarios similares a los seguidores de una cuenta especifica.

**Cuentas utiles para Espana (por sector):**

| Sector | Cuentas ejemplo |
|---|---|
| SaaS/Tech | @ProductHunt, @TheFamily_es, @Seedrocket |
| Marketing | @Marketingdirecto, @PuroMarketing, @Brandwatch |
| Finanzas | @Rankia, @Finect, @ValueSchool |
| Emprendimiento | @Endeavor_Spain, @Startupxplore |
| B2B | Competidores directos, lideres del sector |

**Regla:** incluir 5-10 cuentas por grupo de anuncios. Mezclar competidores + influencers + medios del sector.

### 6. Tailored audiences

| Tipo | Fuente | Uso |
|---|---|---|
| **Customer list** | Emails, telefonos, user IDs | Retargeting CRM, exclusion clientes |
| **Website visitors** | X Pixel | Retargeting web (usuarios que visitaron pero no convirtieron) |
| **App users** | Mobile Measurement Partner | Retargeting app |
| **Engagement** | Usuarios que interactuaron con tus ads/posts | Warm retargeting |
| **Lookalike** | Basado en cualquiera de los anteriores | Expansion a usuarios similares |

**Match rate estimado (Espana):** 30-50% para listas de email. Mayor si incluyes telefono (+34).

### 7. AI-Optimized Targeting

- X muestra anuncios mas alla de tu segmentacion manual si predice mejor conversion
- **Activar despues** de tener 50+ conversiones (necesita datos)
- Monitorizar que no se desvie hacia audiencia irrelevante

## Estrategias de audiencia por objetivo

### Lead gen B2B

```
Grupo A: Keywords de intencion ("necesito [solucion]", "mejor [categoria]")
Grupo B: Follower look-alikes de competidores
Grupo C: Retargeting website visitors (Pixel)
```

### Lead gen servicios locales

```
Grupo A: Keywords locales + interes ("[ciudad] + servicio")
Grupo B: Geo-targeting 30km + intereses relevantes
Grupo C: Retargeting website visitors
```

### SaaS signup / free trial

```
Grupo A: Keywords de pain point ("automatizar X", "problemas con Y")
Grupo B: Lookalike de clientes actuales
Grupo C: Retargeting visitantes pagina pricing
```

## Exclusiones — criticas para eficiencia

**Siempre excluir:**
- Clientes actuales (tailored audience de CRM)
- Empleados propios
- Usuarios que ya convirtieron (lista de conversiones)

**Considerar excluir:**
- Menores de 25 si B2B
- Regiones fuera de tu area de servicio
- Keywords de empleo si no contratas
- Competidores (o incluirlos — depende de estrategia)

## Tamano de audiencia recomendado

| Objetivo | Tamano minimo | Optimo |
|---|---|---|
| Awareness | 100.000+ | 500.000-2M |
| Trafico/Engagement | 50.000+ | 200.000-1M |
| Conversiones | 20.000+ | 50.000-500.000 |
| Retargeting | 1.000+ | 5.000-50.000 |

**Audiencia demasiado grande** = gasto ineficiente. **Demasiado pequena** = CPM alto y poca entrega.
