# Framework de Escalado — X Ads Espana

## Cuando escalar

**Prerequisitos antes de escalar:**
- Campana rentable durante 2+ semanas consecutivas
- CPL/CPA estable (variacion <20% semana a semana)
- 30+ conversiones en los ultimos 30 dias
- Tracking completo funcionando (Pixel + CAPI idealmente)

## Escalado vertical (profundizar)

### Aumentar presupuesto gradualmente

- **Regla de oro:** max +20% cada 3-5 dias
- NUNCA duplicar presupuesto de golpe → resetea el algoritmo
- Monitorizar CPA las 72h despues de cada aumento

### Secuencia de aumento

```
€50/dia → €60 (dia 5) → €72 (dia 10) → €86 (dia 15) → €100 (dia 20)
```

Asi llegas de €50 a €100 en ~20 dias sin desestabilizar.

### Expandir el creativo ganador

Cuando un creativo funciona:
1. Crear 3-4 variaciones del mismo angulo (diferente hook, mismo beneficio)
2. Probar mismo copy en diferente formato (texto → imagen → video)
3. Adaptar copy a diferentes sub-audiencias
4. **No tocar el creativo original** — dejarlo corriendo mientras pruebas

### Profundizar keywords ganadoras

- Identificar las 10 keywords con mejor CPL
- Crear grupo de anuncios dedicado a esas keywords
- Anadir keywords relacionadas/sinonimas
- Aumentar puja en keywords top

## Escalado horizontal (expandir)

### Nuevos formatos

Secuencia recomendada de formatos:

```
Texto/Imagen → Video (16:9) → Vertical Video (9:16) → Carousel → Amplify
```

- Cada nuevo formato = nuevo grupo de anuncios
- Presupuesto de testing: 5-10% del gasto total
- Minimo 2 semanas de test antes de evaluar

### Nuevas audiencias

| Expansion | Descripcion |
|---|---|
| Follower look-alikes | Anadir cuentas similares a las ganadoras |
| Nuevos intereses | Intereses adyacentes al nucleo |
| Lookalike audiences | Basado en mejores converters |
| Geo-expansion | Madrid → Barcelona → Valencia → resto |
| Edad expansion | Ampliar rango 5 anos arriba o abajo |

### Nuevos objetivos de campana

Si funciona Awareness → probar Consideration → probar Conversion.

```
Awareness (CPM) → Website Traffic (CPC) → Lead Gen (CPA) → Retargeting
```

### Retargeting (obligatorio al escalar)

Tipos de retargeting en X:

| Audiencia | Ventana | Uso |
|---|---|---|
| Website visitors | 7-30 dias | Usuarios que visitaron pero no convirtieron |
| Engaged users | 7-14 dias | Usuarios que interactuaron con anuncios |
| Video viewers | 7 dias | Usuarios que vieron 50%+ de un video ad |
| Partial converters | 30 dias | Empezaron formulario pero no terminaron |

**Presupuesto retargeting:** 20-30% del total cuando escalas.

## Secuencia de escalado recomendada

### Fase 1: Validacion (mes 1-2)

- 1-2 campanas, 2-3 ad groups
- Presupuesto: €500-1.000/mes
- Objetivo: encontrar audiencia + creativo ganador
- KPI: CPL viable, CTR > 0,8%

### Fase 2: Escala inicial (mes 2-3)

- 2-3 campanas (prospecting + retargeting)
- Presupuesto: €1.000-3.000/mes
- Objetivo: volumen estable de leads
- KPI: CPL estable, 20+ leads/mes

### Fase 3: Escala media (mes 3-6)

- 3-5 campanas (multi-formato, multi-audiencia)
- Presupuesto: €3.000-10.000/mes
- Objetivo: diversificar formatos, full funnel
- KPI: CPL decrece con escala, 50+ leads/mes

### Fase 4: Enterprise (mes 6+)

- 5+ campanas + Amplify + Takeovers ocasionales
- Presupuesto: €10.000+/mes
- Objetivo: dominar share of voice en el nicho
- KPI: brand lift + lead gen combinado

## Errores comunes al escalar

| Error | Consecuencia | Solucion |
|---|---|---|
| Duplicar presupuesto de golpe | CPA sube 50-100% | Incrementos de max 20% |
| No rotar creativos | Fatiga → CPL sube | 5-8 creativos activos, refresh quincenal |
| Escalar sin retargeting | Pierdes 20-30% conversiones warm | Activar retargeting desde fase 2 |
| No excluir converters | Gastas en quienes ya son leads | Tailored audience de exclusion |
| Copiar campana ganadora | Canibalizacion de audiencia | Diferenciar audiencias entre campanas |
| Ignorar estacionalidad | Agosto escalar = quemar dinero | Reducir en agosto, escalar en sept |
