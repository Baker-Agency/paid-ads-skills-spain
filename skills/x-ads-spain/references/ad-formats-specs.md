# Formatos de Anuncio y Especificaciones — X Ads

## Promoted Ads

### Imagen unica

| Spec | Valor |
|---|---|
| Aspect ratio | 1:1 (cuadrado) o 1.91:1 (horizontal) |
| Tamano recomendado | 1200×1200px (1:1) o 1200×628px (1.91:1) |
| Tamano minimo | 800×800px (1:1) o 800×418px (1.91:1) |
| Formato archivo | JPG, PNG |
| Max file size | 3MB |
| Copy | 280 chars max (23 chars por enlace). Recomendado <100 chars |

### Video

| Spec | Valor |
|---|---|
| Aspect ratio | 16:9 (horizontal) o 1:1 (cuadrado) |
| Tamano recomendado | 1280×720px (16:9) o 720×720px (1:1) |
| Formato archivo | MP4, MOV |
| Max file size | 1GB (recomendado <30MB para rendimiento) |
| Duracion | 6-15 segundos optimo (max 140s standard, 4h premium) |
| Audio | Recomendado pero con subtitulos obligatorios |
| Thumbnail | Automatico o personalizado |

**Reglas de video:**
- Hook en primeros 3 segundos
- Subtitulos siempre (80%+ ven sin sonido en mobile)
- Branding visible desde segundo 1
- CTA visual al final

### Carousel

| Spec | Valor |
|---|---|
| Tarjetas | 2-6 por carousel |
| Aspect ratio | 1.91:1 o 1:1 (mismo en todo el carousel) |
| Puede mezclar | Imagen + Video en un carousel |
| Copy por tarjeta | Titulo independiente por tarjeta |
| CTA | Un CTA por tarjeta + CTA global |

**Mejor para:** mostrar multiples servicios, casos de exito, pasos de proceso.

### Texto puro

| Spec | Valor |
|---|---|
| Copy | 280 chars max |
| Enlace | Opcional (23 chars) |
| Sin imagen/video | Aspecto 100% organico |

**Mejor para:** engagement organico, bajo coste, prueba de messaging.

## Vertical Video Ads

| Spec | Valor |
|---|---|
| Aspect ratio | 9:16 (vertical, pantalla completa) |
| Tamano | 1080×1920px |
| Duracion | 6-15 segundos optimo |
| Sound | On por defecto (pero subtitulos recomendados) |
| Formato | MP4, MOV |

**Datos clave:**
- 7x mas engagement que otros formatos
- 20% mas ventas vs horizontal
- Representa 20% del tiempo diario de usuarios
- **Formato de mayor crecimiento en X**

## X Amplify

### Pre-roll

| Spec | Valor |
|---|---|
| Duracion | 6-15 segundos |
| Skip | Skippable a los 6 segundos |
| Categorias | 15+ (deportes, noticias, entretenimiento, gaming) |
| Compra | Self-serve (Amplify Pre-roll) |

### Sponsorships

| Spec | Valor |
|---|---|
| Formato | Video dedicado con publisher especifico |
| Compra | NO self-serve (contactar equipo X Ads) |
| Presupuesto | Premium |

## Takeovers

### Timeline Takeover

- Primer anuncio que ve el usuario al abrir X
- Formato: imagen, video o GIF
- Compra: reserva diaria (no subasta)
- **Precio premium** — solo para lanzamientos grandes

### Trend Takeover

- Aparece junto a trending topics en tab Explorar
- Formato: hashtag + descripcion (70 chars) + opcional video (Trend Takeover+)
- Compra: reserva diaria
- **Precio premium** — maximo alcance posible

## Collection Ads

| Spec | Valor |
|---|---|
| Hero | 1 imagen o video principal |
| Thumbnails | 2-5 imagenes secundarias |
| Destino | Cada thumbnail puede tener URL distinto |
| Mejor para | Multiples servicios, planes, testimonios |

## Dynamic Product Ads (DPA)

| Spec | Valor |
|---|---|
| Requisito | Catalogo de productos configurado |
| Retargeting | Usuarios que visitaron productos especificos |
| Prospecting | Usuarios similares a compradores |
| Formato | Automatico desde feed |

**Nota para lead gen:** DPA es mas relevante para e-commerce. Para servicios, usar Collection Ads.

## Features adicionales

| Feature | Descripcion |
|---|---|
| **Polls** | Encuestas interactivas en el anuncio |
| **Conversation buttons** | Botones que generan tweet pre-escrito |
| **Website button** | CTA prominente con enlace a web |
| **App button** | CTA para descarga de app |
| **Branded hashtag** | Hashtag personalizado con emoji |
| **Branded notifications** | Usuarios opt-in para recibir recordatorios |

## Resumen: que formato usar

| Objetivo | Formato recomendado |
|---|---|
| Lead gen (formulario) | Promoted Ad imagen + Website button |
| Brand awareness | Vertical Video o Video Ad |
| Engagement | Texto puro o Poll |
| Retargeting | Carousel o Collection |
| Lanzamiento evento | Takeover + Branded Notification |
| Multi-servicio | Carousel o Collection |
| B2B thought leadership | Texto puro (hilo) o imagen con dato |
